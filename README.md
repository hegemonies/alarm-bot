# alarm-bot

Alarm bot for telegram.

## Features

### Create

Create of alarm.

> create <format>

#### Formats

- hours:minutes -> 10:30
- day-month-year hours:minutes -> 10-08-2021 10:30
- day-month hours:minutes -> 10-08 10:30
- cron -> 30 10 * * 1-5

### Delete

Delete of alarm.

> delete <alarm-id>
